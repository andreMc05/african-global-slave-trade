# Global African Slave Trade — Interactive Atlas

An interactive 3D globe exploring forced migration pathways, regional powers, and cultural identities across Africa from 1400–1900 CE.

**[Read the full user guide →](docs/GUIDE.md)**

## Quick start

```bash
python3 -m http.server 8080
```

Open [http://localhost:8080](http://localhost:8080) in your browser.
